# 信息架构（PC Web）

- 登录（01_Login）
- 首页仪表盘（02_Home_Dashboard）
  - 顶部导航：课程 / 通知 / 预约 / 食堂 / 地图 / 报修 / 助手
  - 左侧侧栏：功能树与快捷入口，顶部展示品牌 logo（assets/logo.png）
  - 主区卡片：今日安排、最新通知、快捷操作
- 智能助手（03_Assistant_Chat）
  - 对话区（rpChat）/ 候选建议（rpHints）/ 输入区
- 课程（04_Courses）
  - 周/日视图切换，课程列表（rpCourses），详情浮层
- 通知中心（05_Notices）
  - 分类筛选（教务处/学院/后勤/全部），置顶/未读，详情
- 预约（06_Reservations）
  - 房间列表（rpRooms），预约表单（时间/人数/设备），结果页
- 食堂（07_Canteen）
  - 菜品列表（rpDishes），口味/热量筛选
- 地图（08_Map）
  - POI 列表（rpPOI），起终点选择与路线说明
- 报修/反馈（09_Tickets）
  - 报修单提交，状态流转（rpTickets）
- 个人中心（10_Profile）
  - 主题切换（浅色/深色）、角色切换（学生/教师）
