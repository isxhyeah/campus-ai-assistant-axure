# 基于人工智能的校园智能助手系统（Axure RP 10 / PC Web / 深色模式 / Apple HIG 风格）

本仓库提供 Axure RP 10 原型的完整搭建说明与数据文件，目标是快速产出"校园智能助手"PC Web 原型，遵循苹果 Human Interface Guidelines 的设计理念（清晰、内敛、深度），并支持深浅色主题切换。品牌位采用校徽：assets/logo.png。

交付包含：
- PC 端页面信息架构与交互流程
- HIG 风格样式规范与设计 tokens（含深色模式）
- Repeater 数据源（CSV，可直接导入）
- 仿"AI 回答/思考中/候选建议/多轮意图"逻辑
- Axure 构建步骤（无后端）

## 快速预览
1) 使用 Axure RP 10 打开 prototype/CampusAIAssistant.rp
2) 运行 02_Home_Dashboard -> 03_Assistant_Chat
3) 在右上角切换浅/深色（gTheme）
